一、定义与说明

1.1 个人信息：指以电子或者其他方式记录的能够单独或者与其他信息结合识别自然人个人身份的各种信息，包括但不限于自然人的姓名、出生日期、身份证件号码、个人生物识别信息、住址、粗略位置信息等。

1.2 个人敏感信息：指一旦泄露、非法提供或滥用，可能危害人身和财产安全，极易导致个人名誉、身心健康受到损害或遭受歧视性待遇等的个人信息。例如身份证件号码、个人生物识别信息、银行账号、通信内容、健康生理信息等。为便于您识别，本政策中对敏感信息会以加粗、单独列示等方式重点提示。

1.3 设备：指用于访问空调万能遥控通器服务的装置，例如智能手机、平板电脑等。

1.4 唯一设备标识符（UUID 等）：指由设备制造商编入设备中的一串字符，可用于以独有方式标识相应设备。

1.5 Cookie：指您浏览网页时，网站服务器放在客户端（您的移动电话或其他智能终端）中的文本文件，可用于改善您的使用体验。

二、我们如何收集和使用您的个人信息

我们仅会出于本政策所述目的，收集和使用您的个人信息。

2.2 开展营销活动

空调万能遥控通器会根据运营情况适时推出各类营销活动。您选择参加时，可能需要提供姓名、联系地址、联系方式、银行账号或第三方支付账号信息。其中部分信息可能属于个人敏感信息。若您拒绝提供，可能无法向您发放红包或其他奖励，但不影响您正常使用空调万能遥控通器。

2.3 改善产品或服务

为改善用户体验，我们可能会获取您的客服反馈信息、您主动提供的联系方式、设备安装列表等信息，并在必要范围内进行关联分析，用于优化产品功能与服务质量。

在以下情形中，我们收集、使用个人信息无需另行征得您的授权同意：

  1. 与国家安全、国防安全有关的；
  2. 与公共安全、公共卫生、重大公共利益有关的；
  3. 与犯罪侦查、起诉、审判和判决执行等有关的；
  4. 出于维护个人信息主体或其他个人生命、财产等重大合法权益但又很难得到本人同意的；
  5. 所收集的个人信息是个人信息主体自行向社会公众公开的；
  6. 从合法公开披露的信息中收集的您的个人信息，例如合法新闻报道、政府信息公开等渠道；
  7. 根据您的要求签订合同所必需的；
  8. 用于维护所提供产品或服务的安全稳定运行所必需的，例如发现、处置产品或服务故障；
  9. 法律法规规定的其他情形。

请注意，单独或与其他信息相结合后仍无法识别您的身份或与您建立联系的信息，不属于个人信息；但如果该等信息与其他信息结合后可识别到您，则在结合使用期间，将被视为个人信息并按本政策处理。

2.4 我们获取的权限对应的业务功能

我们仅会出于本政策所述目的获取相应手机权限。相关权限通常会通过弹窗等方式向您提示并由您授权；不同终端设备上的权限获取方式可能不同，以您设备的实际展示为准。部分权限可能由第三方 SDK 获取，详见本政策第四条。

设备标识权限（含 IMEI、MAC、OAID、Android ID、SN、设备 IP）

详细范围：可读取设备唯一标识类信息与网络标识信息，支持在 App 前台、后台及静默状态下获取；其中 OAID 可能在同意隐私政策后随 App 启动请求。

收集目的：用于统计分析、风控、异常日志分析、功能联动和性能优化。

数据处理方式：相关信息仅用于 App 自身的统计、风控与功能联动，不直接追溯个人身份。

WiFi 信息权限（含 SSID、BSSID、WiFi 扫描结果）

收集目的：用于检测网络状态、优化连接质量、提升体验及辅助安全风控。

数据处理方式：仅用于网络连接、性能提升与风控，不用于额外的个人信息画像，也不会与第三方共享。

存储权限

收集目的：用于保存 App 运行过程产生的缓存数据、下载用户所需内容，以及读取与功能相关的本地资源。

数据处理方式：仅操作与 App 功能相关的数据，不随意读取用户私人存储内容。

麦克风权限

收集目的：用于语音相关工具、意见反馈中的语音留言等功能。

数据处理方式：录音文件仅用于当前功能场景，不随意存储或与第三方共享。

相机与相册权限

收集目的：用于意见反馈拍照上传、工具功能、扫码、素材拍摄、相册图片/视频上传及素材选择等。

数据处理方式：拍摄或读取的图片、视频仅用于当前操作功能，不随意存储或与第三方共享。

音频文件读取权限

收集目的：用于播放本地音频文件，保障音频相关功能正常运行。

数据处理方式：仅读取与 App 功能相关的音频文件，不读取无关私人音频内容。

收集目的：用于根据应用变化适配联动功能、移除已卸载应用的相关服务，保障兼容性。

数据处理方式：仅读取应用名称、包名等基础信息，不读取应用内部数据。

收集目的：主要用于安全风控及必要的定位辅助。

数据处理方式：仅在必要场景下收集，用户可通过系统权限设置关闭授权。

收集目的：用于增强功能体验、姿态控制、界面适配及性能优化。

三、我们如何使用 Cookie 及相关技术

3.1 Cookie 及同类技术是互联网中普遍使用的技术。当您使用空调万能遥控通器相关服务时，我们可能会向您的设备发送一个或多个 Cookie 或匿名标识符，以收集和存储您访问、使用本产品时的信息。

我们承诺不会将 Cookie 用于本隐私政策所述目的之外的任何其他用途。Cookie 及相关技术主要用于：

  1. 保障产品与服务的安全、高效运转：识别您是否安全登录，判断是否存在盗用、欺诈等异常行为，并提升登录和响应效率。
  2. 帮助您获得更轻松的访问体验：减少重复输入信息，提升使用便捷性。

Cookie 的清除：您可以在浏览器设置中进行数据清除。清除后，您可能无法继续使用依赖 Cookie 的部分服务或功能。

四、我们如何共享、转让、公开披露您的个人信息

4.1 共享和披露

空调万能遥控通器不会将您的个人信息向空调万能遥控通器以外的任何公司、组织和个人共享，但以下情况除外：

  1. 在获得您（及监护人）明确同意、授权或您主动选择的情况下共享；
  2. 根据法律法规、诉讼争议处理需要，或应行政、司法机关依法提出的要求共享；
  3. 向关联方共享必要的个人信息，以实现本政策所述目的；如共享敏感信息或改变使用目的，将再次征求您的授权同意；
  4. 向授权合作伙伴共享为您提供服务所必需的设备信息，以提升客户服务与用户体验。

按照文档描述，合作共享场景包括但不限于以下类型：

广告推送：可能与服务提供商共享 IMEI、MAC 地址、网络类型、经纬度及特定类型传感器信息。

账户登录与提现：可能与腾讯-微信等服务提供商共享设备信息、位置信息及网络状态。

重要信息推送：可能与服务提供商共享网络状态、IMEI、设备存储信息。

反作弊与问题排查：可能与腾讯、同盾等服务提供商共享设备信息、位置信息和网络状态。

我们会与共享对象签署严格的数据保护协议，要求其按照我们的说明、本政策及其他保密和安全措施处理个人信息。

为了您能够正常使用产品，我们可能通过 API、SDK 等与第三方进行必要的数据共享。不同版本接入的第三方 SDK 可能有所不同，通常包括第三方账号登录、厂商推送等能力。
